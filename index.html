    全自動股票管理表

    只需做好交易紀錄，其餘的都幫你搞定～✨

    🧭 免費試用💡 使用指南🪀 試算表預覽🛍️ 訂閱

    簡單 3 步驟立即使用

    3 分鐘即可建立屬於自己的股票管理試算表

    • 步驟1. 填寫免費試用表單

    • 步驟2. 在 Gmail 信箱收信，透過信件內連結複製屬於自己的試算表

    • 步驟3. 啟用並授權

    讓這些功能替你輕鬆管理股票

    自動更新股票成交價

    開啟試算表後會自動更新股票最新成交價(非即時)，並以新的成交價做各種後續計算

    多種視覺化圖表

    輕鬆享受多種圖表化的數據，持股比例、賺賠一目了然，且無須動手，當股票價格有變動時圖表會自動更新

    自動化計算

    自動計算多項數據，交易成本(交易稅、手續費)、各項報酬率(已實現損益、未實現損益)等等

    自動記錄投資變化

    每周五盤後，會將當下的成本、市值、未實現損益等記錄下來。當時間拉長，可以觀察到自己投資的曲線變化！

    輕鬆管理多家券商

    靈活的設計，可以同時管理不同的券商並依照券商不同的折讓手續費各別設定，還能看各券商.分類的占比圖

    一鍵更新

    操作簡單，每次紀錄完新的交易資料後，只需按一次「更新儀表板」按鈕，便可更新各項數據及圖表

    多種百分比圖

    持股 / 產業 / 現金 / 券商 or 分類

    切換資料範圍的區間

    可設定儀表板的資料範圍區間，方便查詢過去的績效損益

    頁面

    強大的功能與多個分頁能滿足多種需求

    • 儀表板

      股票管理表的主頁，包含多種圖表及數據，而且全都自動更新

    • 交易紀錄

      紀錄的主要頁面，分為手動輸入區與自動生成區

    • 現金股利

      包含現金股利的紀錄與報酬率的計算，設有存股計畫的區塊

    • 觀察

      可將觀察中的股票名單記錄在這裡，價格會有醒目的顏色提醒

    • 更多

      持股與歷史、統計、券商與分類、股票代號、自動記錄

    新功能

    更全面的儀表板

    新增許多數字面板與比較值，使用上更加直覺

    圖表的下拉選單

    圖表多了靈活切換數據的選項，可依自己需求選擇顯示的項目 / 順序 / 數量

    資金 / 現金比例

    投入資金與現金的比例，也將現金新增至自動記錄的走勢圖，方便觀察

    新的取價結構

    新設計的架構可將網路取價的失敗機率降至最低

    更精準的計算

    新增 1 元價差的設定

    推薦機制 ( 額外 )

    可藉由推薦親友使用，獲得推薦獎勵的訂閱月數，延長使用期限

    國際版

    • 跨市場總覽

      同步多個國際版，將其整合在一起，方便整體瀏覽

    • 美股收費版

      新增美金匯率，以美金為計算基準，由美股網站取得股票價位

    • 歐股收費版

      新增歐元匯率，以歐元為計算基準，由歐股網站取得股票價位

    • 港股收費版

      新增港幣匯率，以港幣計算基準，由港股網站取得股票價位

    客製化

    依照客戶需求，提供建議並為其設計的個人版本

    • 客製定期定額

      為定期定額做一個分類，有專屬圖表和表格可以看累積獲利(含股利)

    • 客製融資版

      新增融資分頁，專門計算融資券的投資績效

    • 客製權證版

      新增權證代號，可手動輸入權證價格並自動帶入計算

    • 美股客製版

      新增換匯的功能頁面，紀錄美金的交易歷程
